# СКИН
> Система контроля ИТ - направления

## Использование
Для запуска потребуется Git, Docker и Python

В корне репозитория выполняем:
```bash
docker compose up
```
Docker выполнит всю работу и будет запущен веб интерфейс.

## Использование №2
Проект можно использовать и без Docker.

Важно убедиться, что при выполнении команд далее используется виртуальная среда.
В корне репозитория выполняем:
```bash
pip install -r requirements.txt
python src/main.py
```
После этого будет запущен веб-интерфейс.

## Настройка
Настройки подтягиваются из .env файла, который изначально содержит настройки по-умолчанию.
- CACHE_FOLDER - папка относительно корня, куда будет сохранен кеш обработки некоторых данных
- WEB_INTERFACE_PORT - порт на котором будет открыт веб-интерфейс

## Обновление информации
Обновить данные в БД можно из веб-интерфейса (в правом верхнем углу будет кнопка открытия нужного меню).

Также это можно сделать и вручную.
Для обновления вручную выполняем следующую команду в корне репозитория:
```bash
python src/parsing.py
```
После некоторого времени все данные в БД будут обновлены.

## Добавление ресурсов для отслеживания
Добавить ресурсы для отслеживания можно в веб-интерфейсе (правый верхний угол - кнопка открытия меню)
